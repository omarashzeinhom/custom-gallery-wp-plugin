# Custom WordPress Gallery Plugin For PlayList

- Project Structure

```bash
/custom-gallery-plugin
     custom-gallery-plugin.php
     uninstall.php
     /languages
     /includes
     /admin
          /js
          /css
          /images
     /public
          /js
          /css
          /images
```

## Start By

1. Install  Dev Dependencies using `yarn`.

```bash
npm install -g yarn
yarn install
```

2. Start Development Server 

```bash
gulp
#or 
npx gulp
```

3. Start NodeLiveReload

```bash
node livereload.js
```
